**Side Scroll** adalah game **2D side-scrolling** di mana pemain berperan sebagai sebuah karakter yang harus **melewati dan menghindari para hantu** untuk mencapai tujuan.  
Game ini berfokus pada **ketangkasan pemain**, timing, dan refleks dalam menghadapi rintangan yang muncul sepanjang permainan.

---

## ğŸ® Gameplay

- Pemain mengontrol satu karakter utama
- Karakter akan bergerak dari **kiri ke kanan (side scroll)**
- Sepanjang perjalanan, pemain akan menghadapi **musuh berupa hantu**
- Pemain harus **menghindari atau melewati hantu**
- Jika karakter terkena hantu, maka permainan akan berakhir (*Game Over*)
